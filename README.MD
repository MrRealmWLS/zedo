## About Project

ZEDO is a comprehensive Discord utility tool designed for advanced users. It provides various functionalities to manage Discord accounts and interactions, including token validation, server management, and promotional code checking. The tool is built with Python and offers a console-based interface with colorful output for enhanced user experience.

## Features

- **Promo Checker**: Validate Discord promotional codes from a file and save valid ones
- **Token Info**: Get detailed information about a Discord token including username, Nitro status, verification status
- **Remove Friends**: Bulk remove all friends from a Discord account
- **WebHook Spam**: Send multiple messages to a Discord webhook rapidly
- **Leave All Guilds**: Automatically leave all servers associated with an account
- **Webhook Delete**: Delete a Discord webhook permanently
- **Token Checker**: Verify if a Discord token is valid
- **Clear Screen**: Clean the console interface
- **Webhook Info**: Retrieve information about a Discord webhook
- **Close DM**: Close all direct message conversations
- **Credits**: View project credits and information

## Installation

1. Make sure you have Python 3.x installed on your system
2. Install the required dependencies:
   ```bash
   pip install requests aiohttp colorama
   ```
3. Download the project files
4. Run the main script:
   ```bash
   python main.py
   ```

## Requirements

- Python 3.x
- requests
- aiohttp
- colorama

## Usage

Run the program and select from the available options:
- Enter option number to execute desired function
- Follow the on-screen prompts for each feature
- Use option 12 to exit the application
